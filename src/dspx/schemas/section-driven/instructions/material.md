# 寫 material.md（寫作素材，可選）

material 是 draft 寫稿的**血肉原料倉**，不是第二份成稿。消化過的來源/事實/例子放這裡，draft 取材於此。血肉薄的章節可以沒有 material（concept + decisions 就夠）。

格式（lint 會驗）：
- 每塊一個標題：`## <類型>: <標題> {#m-<錨>}`
- 類型限：`src`（來源摘錄）/ `fact`（事實）/ `framing`（框架）/ `eg`（例子）/ `layout`（版面提示）。
- 塊內**只用條列 / 引文 / 表格**，不要寫成段散文（散文是 docs 成品的事，避免變成第二份草稿）。
- 不要留雜思痕跡（`🟡`、否決標）——material 只收消化後的客觀事實（缺的事實用 `[TBD]` 標）。
